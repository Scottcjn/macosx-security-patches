# Contributing to macOSX Security Patches

Security patches for legacy Mac OS X (Panther, Tiger, Leopard, Snow Leopard).

## Development Setup

```bash
git clone https://github.com/MingYu5/macosx-security-patches.git
cd macosx-security-patches
```

## Supported Systems

- Mac OS X Panther (10.3)
- Mac OS X Tiger (10.4)
- Mac OS X Leopard (10.5)
- Mac OS X Snow Leopard (10.6)

## Contributing

1. Fork the repo
2. Create a feature branch
3. Test on legacy hardware
4. Open PR against `Scottcjn/macosx-security-patches:main`

Questions? Open an issue!
